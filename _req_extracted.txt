Inventory Management System (IMS) - Project Report 
 
1. Project Overview 
Title: 
Multi-Tenant Subscription-Based Inventory Management System 
Objective: 
To design and develop a scalable SaaS-based inventory management system that enables multiple companies to manage their products, stock, and team members efficiently under subscription-based plans. 
Core Idea: 
A centralized platform serving multiple businesses with separate data, controlled access, and subscription-based usage. 
 
2. Key Features 
• Multi-tenant architecture (multiple companies with isolated data) 
• Secure authentication using JWT 
• Product and category management 
• Real-time inventory tracking 
• Low stock alerts 
• Role-based access control (Admin, Staff) 
• Subscription-based feature restrictions 
• Structured and functional dashboard interface

6. Core Modules and Working Flow 
Authentication Module 
   • Company registration and admin creation 
   • Secure login using JWT 
Product Management 
   • Add, edit, delete, and view products 
   • Products are linked to a specific company 
Category Management 
   • Create and manage product categories Inventory Management Stock Types: 
   • Available Stock 
   • Reserved Stock 
   • Sold Stock 
Workflow: 
Order placed ? Stock reserved 
Order shipped ? Stock deducted 
Order cancelled ? Stock restored 
Dashboard Module 
   • Displays inventory insights and key metrics 
User Management 
   • Admin can add, update, or remove users 
   • Role-based permissions (Admin and Staff) 
 
7. Subscription System 
Plans 
Basic Plan 
   • Maximum 50 products 
   • 1 user 
   • Limited features 
Pro Plan 
   • Maximum 500 products 
   • Up to 5 users 
   • Additional features such as alerts 
Business Plan 
   • Unlimited products 
   • Unlimited users 
   • Full feature access 
 
Subscription Workflow

?
